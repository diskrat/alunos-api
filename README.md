# REST API Project

This is a simple REST API built with Node.js. The project demonstrates the use of Express.js for building APIs and includes basic CRUD operations.

## Features

- RESTful API design
- CRUD operations
- JSON-based responses
- Modular and scalable structure

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/diskrat/rest_api_01.git
  cd rest_api_01
  ```

2. Install dependencies:
  ```bash
  npm install
  ```

## Usage

1. Start the development server:
  ```bash
  npm start
  ```

2. The API will be available at `http://localhost:3001`.

## Scripts

- `npm start`: Start the server
- `npm run dev`: Start the server in development mode (if configured with nodemon)
